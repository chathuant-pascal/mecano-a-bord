Logos des assistants IA — Réglages Mécano à Bord
=================================================

Pour afficher les logos dans la section « Connecter votre compte IA » (Réglages),
déposez dans ce dossier des images au format PNG, avec les noms exacts ci-dessous.

Noms de fichiers attendus (minuscules, underscore pour meta_ai) :
  claude.png      — Claude (Anthropic)
  chatgpt.png     — ChatGPT (OpenAI)
  gemini.png      — Gemini (Google)
  mistral.png     — Mistral
  qwen.png        — Qwen (Alibaba)
  perplexity.png  — Perplexity
  grok.png        — Grok (xAI)
  copilot.png     — Copilot (Microsoft)
  meta_ai.png     — Meta AI
  deepseek.png    — DeepSeek

Recommandations :
  - Format : PNG avec fond transparent de préférence
  - Taille : 96×96 px ou 128×128 px (l’app les redimensionne)
  - Les logos sont des marques déposées ; utilisez les visuels officiels
    (kits presse des éditeurs) pour un usage conforme.

Si un fichier est absent, l’app affiche la première lettre du nom (ex. « C » pour Claude).
